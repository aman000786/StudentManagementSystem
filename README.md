# 🎓 Student Management System

A **Core Java** command-line application that helps manage student records efficiently. This project demonstrates the fundamentals of Java programming, including Object-Oriented Programming (OOP), Collections Framework, exception handling, and file handling.

## 📌 Features

* ➕ Add a new student
* 📋 View all students
* 🔍 Search a student by ID
* ✏️ Update student details
* ❌ Delete a student
* 🔤 Sort students by name
* 📊 Sort students by marks
* 🏆 Automatic grade calculation
* 💾 Optional file handling to store student records

## 🛠️ Technologies Used

* Java
* Object-Oriented Programming (OOP)
* Collections Framework (ArrayList)
* Exception Handling
* File Handling (Optional)
* Java Scanner
* Java Comparator

## 🚀 How to Run

1. Clone this repository.
2. Open the project in your preferred Java IDE (IntelliJ IDEA, Eclipse, or VS Code).
3. Compile all Java files.
4. Run `Main.java`.
5. Use the menu to manage student records.

## 📖 Menu Options

1. Add Student
2. Display Students
3. Search Student
4. Update Student
5. Delete Student

7. Sort By Marks
